# PC investment calculator

Very simple UI to calculate "hidden" costs for development and if a PC is worth the investment or if more investment is better.

The HTML file is also hosted directly as GitHub Pages: [https://ragin-lundf.github.io/pc-productivity-cost-calculator/](https://ragin-lundf.github.io/pc-productivity-cost-calculator/).

An article with more description can be found [here](https://ragin.medium.com/why-a-cheap-and-slow-developer-machine-costs-real-money-730f3f7fa1ba?sharedUserId=ragin).
